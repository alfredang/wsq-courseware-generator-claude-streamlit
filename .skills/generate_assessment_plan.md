# Generate Assessment Plan

## Command
`/generate_assessment_plan` or `generate_assessment_plan`

## Navigate
Generate AP/FG/LG/LP

## Keywords
assessment plan, ap document, create ap, generate ap, grading rubric, assessment criteria, assessment schedule, evidence requirements, i need ap, want ap, need assessment plan, make ap, i need an assessment plan

## Description
Help users generate Assessment Plan (AP) documents for WSQ courses.

## Response
I'll take you to **Generate Courseware** now.

Here's what you'll need:
- **Upload** your approved Course Proposal (CP) document
- **Select** "Assessment Plan" from the document type options

The system will generate an Assessment Plan with assessment methods, grading rubrics, evidence requirements, and mapping to learning outcomes.

**Tip:** The AP should align with the learning outcomes defined in your Course Proposal.

## Instructions
When users ask about creating assessment plans or AP documents:

1. **Required Input**: Approved Course Proposal (CP) document
2. **Assessment Plan Contents**:
   - Assessment methods and criteria
   - Grading rubrics
   - Evidence requirements
   - Mapping to learning outcomes
   - Assessment schedule
3. **Process**:
   - Upload approved CP document
   - Select "Assessment Plan" option
   - Configure assessment parameters
   - System generates AP based on CP content
   - Review and download as Word document
4. **Tips**:
   - AP should align with learning outcomes in CP
   - Include clear performance criteria
   - Define evidence collection methods
   - Specify grading standards

## Capabilities
- Generate Assessment Plan (AP) documents
- Map assessments to learning outcomes
- Define grading rubrics and criteria
- Direct users to use the "Generate AP/FG/LG/LP" module in the sidebar

## Next Steps
After generating your Assessment Plan:
- **Generate assessment materials** (SAQ, Case Study, etc.) — say *"create assessment"*
- **Attach assessments to AP** later — say *"add assessment to ap"*
