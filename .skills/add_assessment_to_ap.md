# Add Assessment to AP

## Command
`/add_assessment_to_ap` or `add_assessment_to_ap`

## Navigate
Add Assessment to AP

## Keywords
add assessment to ap, attach assessment, annex assessment, assessment to ap, append assessment, assessment annex, assessment attachment, link assessment, merge assessment, combine assessment with ap, put assessment in ap

## Description
Help users attach generated assessment materials to their Assessment Plan (AP) document.

## Response
I'll take you to **Add Assessment to AP** now.

Here's what you'll need:
- **Upload** your Assessment Plan (AP) document
- **Upload** the assessment files you want to attach (SAQ, PP, CS, etc.)

The system will automatically merge the assessments into the AP as annexes.

## Instructions
When users ask about attaching assessments to assessment plans, adding annexes, or merging assessment files:

1. **Required Input**: Assessment Plan (AP) + assessment files
2. **Supported Assessment Types**: SAQ, PP, CS, PRJ, ASGN, OI, DEM, RP, OQ
3. **Process**:
   - Upload AP document
   - Upload assessment files to attach
   - System merges them as annexes
   - Review and download updated AP
4. **Tips**:
   - Generate assessments first using Generate Assessment module
   - Ensure assessments align with AP methods

## Capabilities
- Attach assessment materials to Assessment Plans
- Support all 9 assessment types as annexes
- Merge documents automatically
- Direct users to use the "Add Assessment to AP" module in the sidebar
