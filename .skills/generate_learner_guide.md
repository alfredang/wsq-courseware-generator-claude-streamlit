# Generate Learner Guide

## Command
`/generate_learner_guide` or `generate_learner_guide`

## Navigate
Generate AP/FG/LG/LP

## Description
Help users generate Learner Guide (LG) documents for WSQ courses.

## Response
Taking you to **Generate AP/FG/LG/LP**. Upload your Course Proposal to generate a Learner Guide.

## Instructions
When users ask about creating learner guides, student guides, or LG documents:

1. **Required Input**: Approved Course Proposal (CP) document
2. **Learner Guide Contents**:
   - Learning content and materials
   - Exercises and activities
   - Self-assessments
   - Key concepts and summaries
   - References and resources
3. **Process**:
   - Upload approved CP document
   - Select "Learner Guide" option
   - Configure content depth and format
   - System generates LG based on CP content
   - Review and download as Word document
4. **Tips**:
   - LG should complement the Facilitator Guide
   - Include clear learning objectives
   - Add exercises for knowledge retention
   - Provide self-assessment checkpoints

## Capabilities
- Generate Learner Guide (LG) documents
- Create comprehensive learning materials
- Include exercises and self-assessments
- Direct users to use the "Generate AP/FG/LG/LP" module in the sidebar
