# Generate Brochure

## Command
`/generate_brochure` or `generate_brochure`

## Navigate
Generate Brochure

## Keywords
brochure, marketing brochure, course brochure, design brochure, create brochure, make brochure, flyer, course flyer, promotional material, broachure, brocure, i need a brochure, want brochure, need brochure, brochure for my course

## Description
Help users create marketing brochures for WSQ courses.

## Response
I'll take you to **Generate Brochure** now.

Here's what you'll need:
- **Upload** your approved Course Proposal (CP) document, **or**
- **Paste a URL** to your course page

The system will automatically extract course details (title, objectives, duration, target audience) and generate a professional marketing brochure.

**Tip:** Make sure your CP is finalised before generating the brochure so all details are accurate.

## Instructions
When users ask about creating brochures, flyers, or marketing materials for courses:

1. **Required Input**: Course Proposal (CP) document or course URL
2. **Brochure Contents**:
   - Course title and overview
   - Key learning outcomes
   - Target audience
   - Duration and schedule
   - Entry requirements
   - Organisation branding
3. **Process**:
   - Upload CP document or paste course URL
   - System extracts course information
   - Brochure is generated with professional layout
   - Review and download as HTML/PDF
4. **Tips**:
   - Use a finalised CP for accurate content
   - Brochure will use organisation branding from settings
   - Review generated brochure for accuracy before distribution

## Capabilities
- Generate professional course marketing brochures
- Extract course details from CP documents or URLs
- Apply organisation branding automatically
- Direct users to use the "Generate Brochure" module in the sidebar
