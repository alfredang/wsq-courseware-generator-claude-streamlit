# Generate Facilitator Guide

## Command
`/generate_facilitator_guide` or `generate_facilitator_guide`

## Navigate
Generate AP/FG/LG/LP

## Description
Help users generate Facilitator Guide (FG) documents for WSQ courses.

## Response
Taking you to **Generate AP/FG/LG/LP**. Upload your Course Proposal to generate a Facilitator Guide.

## Instructions
When users ask about creating facilitator guides, trainer guides, or FG documents:

1. **Required Input**: Approved Course Proposal (CP) document
2. **Facilitator Guide Contents**:
   - Teaching instructions
   - Delivery methods
   - Timing and pacing
   - Resources needed
   - Activity instructions
   - Discussion points
3. **Process**:
   - Upload approved CP document
   - Select "Facilitator Guide" option
   - Configure instructional methods
   - System generates FG based on CP content
   - Review and download as Word document
4. **Tips**:
   - FG should complement the Learner Guide
   - Include detailed timing for each section
   - Add facilitator notes and tips
   - Specify required materials and resources

## Capabilities
- Generate Facilitator Guide (FG) documents
- Create detailed teaching instructions
- Include activity and discussion guidance
- Direct users to use the "Generate AP/FG/LG/LP" module in the sidebar
