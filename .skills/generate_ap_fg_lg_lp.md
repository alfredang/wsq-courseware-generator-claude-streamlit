# Generate AP/FG/LG/LP

## Command
`/generate_ap_fg_lg_lp` or `generate_ap_fg_lg_lp`

## Navigate
Generate AP/FG/LG/LP

## Description
Help users generate courseware documents including Assessment Plan, Facilitator Guide, Learner Guide, and Lesson Plan.

## Response
Taking you to **Generate AP/FG/LG/LP**. Upload your Course Proposal and select the documents you want to generate.

## Instructions
When users ask about creating courseware, training materials, facilitator guides, or lesson plans:

1. **Required Input**: Approved Course Proposal (CP) document
2. **Available Documents**:
   - **Assessment Plan (AP)**: Assessment methods, criteria, grading rubrics, evidence requirements
   - **Facilitator Guide (FG)**: Teaching instructions, delivery methods, timing, resources needed
   - **Learner Guide (LG)**: Student materials, learning content, exercises, self-assessments
   - **Lesson Plan (LP)**: Session-by-session breakdown, activities, timing, materials
3. **Process**:
   - Upload approved CP document
   - Select which documents to generate (can select multiple)
   - Configure options (e.g., number of days, instructional methods)
   - System generates documents based on CP content
   - Review and download as Word documents
4. **Tips**:
   - Generate all documents together for consistency
   - AP should align with learning outcomes in CP
   - FG and LG should complement each other
   - LP timing should match course duration

## Capabilities
- Assessment Plan (AP) - outlines assessment methods and criteria
- Facilitator Guide (FG) - instructor teaching guide
- Learner Guide (LG) - student learning materials
- Lesson Plan (LP) - detailed session-by-session teaching plan
- Direct users to use the "Generate AP/FG/LG/LP" module in the sidebar
