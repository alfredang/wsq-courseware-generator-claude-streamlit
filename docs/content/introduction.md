# Introduction

Welcome to the **WSQ Courseware Generator** documentation. This platform is a comprehensive AI-powered solution built with the **OpenAI Agents SDK** and **Streamlit**.

## Purpose

The system is designed to automate the creation of educational documents for **Workforce Skills Qualification (WSQ)** training programs. It leverages an orchestrator-based multi-agent architecture to ensure high-quality, validated outputs across various document types.

## Key Outcomes

- **Efficiency**: Reduce document preparation time from days to minutes.
- **Consistency**: Maintain a high standard of quality and formatting across all courseware.
- **Intelligence**: Use state-of-the-art LLMs (GPT-4, Claude, DeepSeek) via a unified interface.

## Core Documents Supported

- **Course Proposal (CP)**
- **Assessment Plans (AP)**
- **Facilitator Guides (FG)**
- **Learning Guides (LG)**
- **Lesson Plans (LP)**
- **Marketing Brochures**
- **Assessment Papers (SAQ, Case Studies, Practical Performance)**
